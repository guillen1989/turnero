{{ _('Cómo funciona Turnero') }}

{{ _('Todo lo que puedes hacer con Turnero, paso a paso.') }}

🗓️

{{ _('1. Descubre cambios en el calendario') }}

{{ _('El calendario te muestra de un vistazo, día a día, qué turnos ofrecen o necesitan tus compañeros.') }} {{ _('Tres modos, un selector arriba:') }}

  • {{ _('Ofertas') }} — {{ _('turnos que otros compañeros trabajarían.') }}
  • {{ _('Peticiones') }} — {{ _('turnos que otros compañeros quieren librar.') }}
  • {{ _('Juntes de noches') }} — {{ _('noches de compañeros que buscan complementar su junte.') }}

{{ _('Toca un día para ver qué franjas hay disponibles y acceder a la publicación completa.') }} {{ _('Si el día está vacío, puedes publicar tu propio cambio directamente desde ahí.') }}

📢

{{ _('2. Publica tu cambio') }}

{{ _('Dile a tus compañeros qué turno quieres ceder y cuál te interesa a cambio.') }} {{ _('Puedes pedir una fecha y turno concreto, o dejar abierto cualquier turno.') }}

{{ _('Hay cinco tipos de publicación:') }}

  • {{ _('Cambio') }} — {{ _('cedes un turno (o varios) y pides otro (u otros) a cambio.') }}
  • {{ _('Regalo') }} — {{ _('trabajas un turno sin pedir nada a cambio. Ideal si ya no lo necesitas.') }}
  • {{ _('Petición') }} — {{ _('quieres que alguien te cubra un día sin darte ningún turno a cambio, o hacer la devolución a convenir.') }}
  • {{ _('Junte de noches') }} — {{ _('intercambias noches sueltas de una semana para juntarlas y librar días consecutivos.') }}
  • {{ _('Cambio de turno en el día') }} — {{ _('por ejemplo, cambiar tu turno de mañana por el de tarde el día 16.') }}
📅

{{ _('3. Publica tu planilla mensual') }}

{{ _('Publica tu cuadrante del mes, con tus turnos y libranzas.') }} {{ _('Al publicarla, tus compañeros pueden saber si estás disponible para hacerles el cambio que buscan, y tú puedes ver quién libra o tiene turno compatible el día que quieres ceder.') }}

{{ _('Rellenar el mes es rápido gracias a tres herramientas:') }}

  • {{ _('Relleno por rango') }} — {{ _('elige "del día X al día Y" y el turno o estado, y se rellenan todos los días de ese bloque de un solo clic. Ideal para semanas de vacaciones o bloques de mañanas consecutivos.') }}
  • {{ _('Selección múltiple') }} — {{ _('marca los días que quieras con los checkbox de cada fila; aparece una barra en la parte inferior para aplicar el mismo turno o estado a todos a la vez. Perfecto para las mañanas o noches sueltas repartidas por el mes.') }}
  • {{ _('Rellenar días vacíos') }} — {{ _('te permite asignar de un solo clic el mismo estado (por ejemplo, Libre) a todos los días que aún no tienen nada.') }}
🤝

{{ _('4. Si hay match, te avisamos') }}

{{ _('Cuando otra persona de tu unidad publique un cambio compatible con el tuyo, recibirás una notificación.') }}

{{ _('Revisas el match en tu panel de "Mis cambios" y') }} {{ _('lo confirmas o lo rechazas') }}. {{ _('El cambio sólo se cierra cuando todos los implicados lo confirman.') }}

{{ _('El detector va más allá del cruce directo entre dos personas:') }} {{ _('también encuentra') }} {{ _('cambios a tres bandas') }} {{ _('(A da a B, B da a C, C da a A) y') }} {{ _('match parciales') }} {{ _('(se resuelve parte de un cambio múltiple aunque el resto quede pendiente).') }}

🔍

{{ _('5. Consulta lo que publican tus compañeros') }}

{{ _('En "Buscar cambios" ves todas las publicaciones abiertas de tu unidad.') }} {{ _('Puedes filtrar por mes, día o turno.') }}

{{ _('Guarda tus filtros favoritos como') }} {{ _('búsquedas guardadas') }} {{ _('y activa las notificaciones: te avisaremos cada vez que aparezca una publicación que encaje con esa búsqueda.') }}

🔔

{{ _('6. Suscríbete a un compañero') }}

{{ _('¿Hay alguien con quien te resulte fácil cambiar? Suscríbete a sus publicaciones y te avisaremos cada vez que publique algo nuevo.') }}

📲

{{ _('7. Comparte tu cambio por WhatsApp') }}

{{ _('¿Conoces a alguien que puede encajarte pero aún no usa Turnero? Desde cualquier publicación tuya puedes') }} {{ _('compartir el enlace directo por WhatsApp') }}.

{{ _('La persona que lo recibe ve los detalles del cambio y puede registrarse en el momento para responderlo.') }}

📝

{{ _('8. Formaliza el cambio con una hoja digital') }}

{{ _('Cuando el cambio esté confirmado, genera la hoja de cambio digital para que la supervisora la autorice desde la app: sin papel, sin idas y vueltas.') }}

📲

{{ _('9. Instala Turnero en tu móvil') }}

{{ _('Puedes instalar Turnero como una app normal en tu móvil: sin pasar por la App Store ni Google Play, directamente desde el navegador.') }} {{ _('Una vez instalada, abre como cualquier otra app, sin barras del navegador.') }}
